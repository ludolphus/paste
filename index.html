About Paste

Paste allows you to create and view short text clippings using app.net. [more]

Paste is a pasteboard/pinboard running on the social networking platform app.net. Each paste has a public link that can be viewed without logging in to app.net, as well as a "private" link which will display both the selected paste and a list of the logged-in user's recent pastes. To create a new paste, you will need a (free) app.net account.

By using Paste you agree to our Terms of Service and Privacy Policy.

Paste is the brainchild of Jonathon Duerig. This version is maintained and hosted by M. C. DeMarco. For more information and help, see the support page.
